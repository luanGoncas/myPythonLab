# 2. Qual das seguintes opções é uma variável e qual é uma string?
- spam Variável
- 'spam' String

# 3. Nomeie três tipos de dados.
- Inteiro, string e ponto flutuante

# 4. Do que é composta uma expressão? O que todas as expressões fazem?
- Uma expressão consiste de valores e operadores que podem retornar um único valor

# 5. Este capítulo introduziu instruções de atribuição, como spam = 10. Qual é a diferença entre uma expressão e uma instrução?
- Uma instrução executa um comando sem necessariamente retornar um valor, enquanto que uma expressão busca sempre retornar um valor único.

# 6. O que a variável bacon contém após a execução do seguinte código?
bacon = 20
bacon + 1
- O valor atribuído à variável bacon é exibido somado com o valor 1, ou seja, exibe o valor 21

spamspamspam

# 8. Por que "eggs" é um nome de variável válido, enquanto "100" é inválido?
- Pois o nome de uma variável em Python não deve começar com números

# 9. Quais três funções podem ser usadas para obter a versão inteira, de ponto flutuante ou de string de um valor?
- int(), float() e str()

# 10. Por que essa expressão causa um erro? Como você pode corrigi-la?
'I eat ' + 99 + ' burritos.'
- Há um valor inteiro na expressão, e não é possível concatenar valores do tipo string com valores do tipo int. Para corrigir, a expressão deve ficar da seguinte maneira:
'I eat ' + '99' + ' burritos.'
